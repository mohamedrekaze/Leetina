# bin:
### clean_cache.sh
- a cleaner for 42 network devices from [Omar BOUYKOURNE](https://github.com/ombhd) on [Github](https://github.com/ombhd/Cleaner_42)
    - Trash contents.
    - 42 caches from Library and Home directories.
    - browsers caches.
    - VSCode Caches and workspaces cache storage.
    - Slack and Discord caches.
    - FileSystems caches in browsers, liek Chrome, Brave and spotify.
    - and other type of caches.
### install.sh
- install's francinette.
    - downloading francinette and installing it.
    - creating a python venv.
    - activate venv.
    - install requirements.
    - print help.
    - replace the current shell with a new one.
### sync_utils.sh
- automating and syncing *utils* directory in test directories
    - setting up the script's directory
    - navigate to the base directory
    - handling a force flag
    - updating utils function
    - syncing util in multiple direcotories
### update.sh
- update and setup of *francinette*
    - OS check
    - navigation to francinette directory
    - update git repo
    - activate venv
    - install requirements
    - final messages
